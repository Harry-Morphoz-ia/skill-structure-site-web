# Photos de la galerie et de la section "À propos"

Dépose ici tes photos avec **ces noms exacts** ; le site les affiche automatiquement.

| Fichier | Où il apparaît | Dimensions | Poids |
|---------|----------------|------------|-------|
| `apropos.jpg` | Section "À propos" | 1200 × 900 px | < 250 Ko |
| `01.jpg` | Galerie, case 1 | 800 × 600 px | < 200 Ko |
| `02.jpg` | Galerie, case 2 | 800 × 600 px | < 200 Ko |
| `03.jpg` | Galerie, case 3 | 800 × 600 px | < 200 Ko |

Pour ajouter plus de cases : dupliquer un bloc `<figure class="tile">` dans `index.html`
et déposer `04.jpg`, `05.jpg`, etc.

Tant qu'un fichier manque, sa case affiche un cadre "Déposez ..." au lieu de casser la page.
Compresser les images sur squoosh.app avant de les déposer.
