# Vidéos

Dépose ici une vidéo nommée `video.mp4` si le site en utilise une.

| Critère | Recommandation |
|---------|----------------|
| Nom | `video.mp4` |
| Format | MP4 (H.264) |
| Poids | **moins de 5 Mo** (au-delà, préférer un lien YouTube/Vimeo) |
| Durée | 10 à 30 s pour une vidéo de fond |

Astuce : pour une vidéo longue (présentation, témoignage), ne pas l'héberger ici.
Intégrer plutôt le lecteur YouTube/Vimeo, plus léger et mieux pour le référencement.
