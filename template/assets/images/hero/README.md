# Image du hero (grande image d'accueil)

Dépose ici **un seul fichier nommé exactement `hero.jpg`**.
Le site l'affiche automatiquement en fond de la première section, rien d'autre à faire.

| Critère | Recommandation |
|---------|----------------|
| Nom | `hero.jpg` (ou `hero.webp`, adapter le CSS) |
| Dimensions | 1920 × 1080 px (paysage) |
| Format | WebP de préférence, sinon JPG |
| Poids | **moins de 300 Ko** (compresser sur squoosh.app) |

Tant qu'aucun fichier n'est déposé, le hero affiche un joli dégradé par défaut.
